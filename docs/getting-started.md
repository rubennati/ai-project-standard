# Getting Started

## 1) Understand repository purpose

Read `README.md` to understand scope and constraints.

## 2) Review contribution and workflow rules

Read:
- `CONTRIBUTING.md`
- `docs/git-workflow.md`
- `AGENTS.md`

## 3) Initialize AI collaboration context

Read:
- `.ai/index.md`
- `.ai/state.md`
- `.ai/routing.md`

## 4) Make a focused contribution

Use a short-lived branch, apply small scoped changes, and open a pull request for review.
